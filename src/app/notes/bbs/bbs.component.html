Bulletin Board Systems
This is a computer server running software that allows users to connect to the system using a terminal program. Once logged in, the user can perform functions such as uploading and downloading software and data, reading news and bulletins, and exchanging messages with other users through email, public message boards, and sometimes via direct chatting.
Many BBSes also offer online games in which users can compete with each other, and BBSes with multiple phone lines often provide chat rooms, allowing users to interact with each other.
