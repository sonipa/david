frameworks & libraries
When you're making an app, libraries help speed things up by providing features you don't have to code yourself. When a library falls out of fashion, usually it can be replaced with something else.
A framework goes further, allowing you to start an app with many of the difficult decisions already made. It's like you're 30% done on day one. From that day forward, you have to work within the framework. When a framework falls out of fashion you get to start over.
